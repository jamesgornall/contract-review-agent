8. CALDICOTT PRINCIPLES COMPLIANCE

All resources must adhere to 8 Caldicott Principles:
1. Justify the purpose for using confidential information
2. Only use patient data when absolutely necessary
3. Use minimum necessary patient data
4. Access on strict need-to-know basis
5. Everyone with access understands responsibilities
6. Comply with the law
7. Duty to share information for care (appropriate sharing)
8. Inform patients about data use

Training: Mandatory within first week of assignment
Refresher: Annual
