PROFESSIONAL SERVICES AGREEMENT
Contract Number: PS-2024-001
Contract Type: Fixed Deliverable

This Professional Services Agreement ("Agreement") is entered into as of January 15, 2024, between Qodea Professional Services Ltd ("Service Provider") and FinTech Innovation Corp ("Client").

1. PROJECT SCOPE AND DELIVERABLES

Service Provider shall deliver a Cloud Migration Platform for Client's banking applications with the following deliverables:

Phase 1 - Discovery and Assessment (Weeks 1-4)
- Current infrastructure assessment report
- Cloud readiness evaluation
- Migration strategy document
- Risk assessment and mitigation plan
- Estimated total cost of ownership (TCO) analysis

Phase 2 - Architecture Design (Weeks 5-10)
- Target cloud architecture design
- Security and compliance framework
- Data migration strategy
- Disaster recovery and business continuity plan
- Performance optimization recommendations

Phase 3 - Implementation (Weeks 11-22)
- Development environment setup
- Application containerization
- Database migration
- Integration with existing systems
- Security hardening and compliance implementation

Phase 4 - Testing and Go-Live (Weeks 23-26)
- User acceptance testing coordination
- Performance testing and optimization
- Production deployment
- Knowledge transfer sessions
- Post-go-live support (2 weeks)

Payment Schedule:
- 20% (£90,000) upon contract signing

- 25% (£112,500) upon completion of Phase 2

- 10% (£45,000) upon successful go-live and sign-off

All payments due within 30 days of invoice date.
