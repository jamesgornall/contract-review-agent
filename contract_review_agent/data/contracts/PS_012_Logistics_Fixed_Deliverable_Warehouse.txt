PROFESSIONAL SERVICES AGREEMENT
Contract Number: PS-2024-012

Client: National Logistics Group Ltd
Project: Warehouse Management System Modernization
Contract Date: September 1, 2024

PROJECT SCOPE

Replacement of legacy warehouse management system across 12 UK distribution centers with cloud-based WMS integrated with existing ERP and transportation management systems.

DELIVERABLES

Phase 1 - Discovery and Planning (Weeks 1-6)
- Current state assessment across 12 sites
- Warehouse operations process mapping
- Integration requirements (SAP ERP, TMS, carrier systems)
- Data migration strategy (10+ years historical data)
- Pilot site selection and readiness assessment
- Implementation roadmap and wave planning

Phase 2 - Core System Build (Weeks 7-16)
- Cloud WMS platform configuration
- SAP ERP integration (inbound/outbound orders)
- TMS integration for shipping
- Carrier API integrations (DPD, DHL, Royal Mail, Hermes)
- Mobile picking application (RF scanners and tablets)
- Inventory management workflows
- Returns processing automation
- Reporting and analytics dashboards

Phase 3 - Pilot Implementation (Weeks 17-24)
- Pilot site deployment (Manchester DC)
- Data migration and validation
- User training (50 warehouse staff)
- Parallel running (2 weeks)
- Go-live support (on-site, 2 weeks)
- Performance tuning and optimization
- Lessons learned and refinement

Phase 4 - Rollout (Weeks 25-44)
- Wave 1: 4 sites (Weeks 25-32)
- Wave 2: 4 sites (Weeks 33-40)
- Wave 3: 3 sites (Weeks 41-44)
- Training delivery (400+ users)
- Data migration for all sites
- Hypercare support per site (1 week each)
- Knowledge transfer and handover
