PROFESSIONAL SERVICES AGREEMENT
Contract Number: PS-2024-007
Contract Type: Fixed Deliverable

This Professional Services Agreement ("Agreement") is entered into as of April 15, 2024, between Qodea Professional Services Ltd ("Service Provider") and Manchester Royal NHS Foundation Trust ("Client").

1. PROJECT SCOPE AND DELIVERABLES

Service Provider shall deliver Electronic Patient Record (EPR) System Integration services connecting legacy systems with the new Epic EPR platform.

Phase 1 - Integration Design and Planning (Weeks 1-6)
- Current state systems integration mapping
- HL7 FHIR interface specifications
- Integration architecture design
- Data migration strategy and risk assessment
- Clinical safety assessment (DCB0129, DCB0160 compliance)
- Security and information governance framework

Phase 2 - Development and Configuration (Weeks 7-18)
- HL7/FHIR interface development (12 priority integrations)
- Legacy system adapters (PAS, PACS, Lab, Pharmacy)
- Master Patient Index (MPI) integration
- Clinical portal single sign-on
- Interoperability testing environment setup

Phase 3 - Testing and Validation (Weeks 19-24)
- Integration testing with all connected systems
- End-to-end workflow testing
- Clinical validation sessions
- Performance and load testing
- Security penetration testing
- Clinical safety testing and documentation

Phase 4 - Deployment and Stabilization (Weeks 25-28)
- Production deployment and cutover
- Hypercare support (2 weeks, 24/7)
- Clinical user training sessions
- Post-go-live monitoring and optimization
- Final documentation and handover
